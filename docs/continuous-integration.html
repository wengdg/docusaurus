Continuous Integration

We use Semaphore CI for RadGrad Continuous Integration. Each time someone commits to the master branch of the RadGrad datamodel GitHub repo, Semaphore will clone this branch, install Meteor, invoke both unit and integration tests, and build the JSDocs.
